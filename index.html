Feature Arsenal

Encrypted Notes

AES-256-GCM authenticated encryption. Per-note Argon2id salt. Notes are encrypted client-side before hitting the database — a wrong password produces a GCM authentication failure, not corrupt data.

AES-256-GCMArgon2idPer-record salt
Encrypted File Vault

Files are metadata-stripped, SHA-256 hashed, then encrypted before any byte reaches the database. Decryption triggers integrity re-verification. The vault password never touches the network.

Blob storageIntegrity check512 MiB max
Secure Share Links

Files are re-encrypted under a share-specific Argon2id key. Recipients never need the vault password. Tokens are 32-byte crypto/rand hex. Shares are time-limited, download-capped, and instantly revocable.

Re-encryptedTTL expiryRevocable
Metadata Stripping

JPEG: binary-level APP1–APP15 segment removal — zero quality loss, no re-encode. PNG: lossless re-encode strips all ancillary chunks. PDF: Info dictionary scrub removes author, timestamps, and creation metadata.

EXIF removalXMP/IPTCPDF Info dict
File Integrity Checker

SHA-256 and SHA-512 computed in a single-pass io.MultiWriter — one read, two hashes. Fingerprints stored in PostgreSQL. Re-hash at any time to detect tampering, corruption, or silent modification.

SHA-256SHA-512Single-pass
Browser-Based UI

Full-stack Go binary embeds the entire SPA. No Node.js, no npm, no separate frontend server. Run secvault serve and open localhost:8080. Deployable to Render with a single render.yaml.

Embedded SPAGo HTTPRender-ready

// THE SOURCE CODE — BATCH SCRIPT TOOLKIT

A collection of advanced Windows batch scripts engineered for systematic data collection, forensic documentation, and authorized system migration. Designed for environments where the operator already holds legitimate administrative rights.

all_drives.bat

Iterates every drive letter A: through Z:, copying data into an organized SRC\Drives\[DriveLetter] hierarchy. Includes comprehensive capacity warnings and automatic abort when the destination drive reaches capacity.

Drive DetectionStructured OutputCapacity Guard
all_public.bat

Targets shared and public directories — C:\Users\Public, C:\Shared, C:\Public, C:\Common — and mirrors them into a clean SRC\Public\ and SRC\Shared\ output structure.

Public FoldersShared DirsOrganized Output
Compilation & Deployment

Scripts are compiled to standalone executables via Advanced BAT to EXE Converter — enabling custom icons, embedded version metadata, and silent execution without an exposed command prompt window for clean operational deployment.
